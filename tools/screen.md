# Screen

## Split

Start with <ctrl-a> then:

> Remember to start a new session in the new terminal with `<ctrl-a>-<c>`.

```
| - split vertically
S - split horizontally
Q - unsplit

tab - switch
space - next
backspace - previous
# - # terminal
" - use list

c - Start new session (bash prompt) in new terminal

<ctrl-a> - switch shells
```
